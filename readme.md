# Endgame | MERN app

Endgame is a web application that allows users to create and manage their goals in an effort to reach their endgame.

## Usage

Rename the .envexample to .env and add your MONGO_URI

### Install dependencies

```
# Backend deps
npm install

# Frontend deps
cd frontend
npm install
```

### Run Server

```
npm run server
```
